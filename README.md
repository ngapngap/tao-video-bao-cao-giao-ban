# Báo Cáo Giao Ban - Video Generator

Ứng dụng Python desktop portable dùng để tạo video báo cáo giao ban từ dữ liệu báo cáo.

## Cài đặt

```bash
pip install -r requirements.txt
```

## Chạy dev

```bash
python app/main.py
```

## Cấu trúc thư mục

```text
app/
  main.py
  ui/
  core/
  pdf/
  ai/
  workflow/
  video/
  security/
  assets/
configs/
tests/
```
